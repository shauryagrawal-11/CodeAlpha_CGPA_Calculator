# CGPA Calculator

## Description
This project is a simple CGPA Calculator developed using C++.

It allows users to enter course details, grades, and credits, then calculates the final CGPA.

## Features
- Input multiple courses
- Enter grade points
- Enter credits
- Calculate total credits
- Calculate final CGPA
- Display course details

## Technologies Used
- C++
- VS Code

## How to Run
1. Open project in VS Code
2. Compile:
   g++ main.cpp -o cgpa
3. Run:
   ./cgpa

## Sample Output
Final CGPA: 8.67
